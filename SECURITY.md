# Security Policy

## Supported Versions

Only the latest release is actively supported. Older versions receive backported critical CVE fixes on a best-effort basis.

## Reporting a Vulnerability

Please report security vulnerabilities privately to `security@divo.ai`. Include:

- A description of the vulnerability
- Steps to reproduce (if available)
- Estimated impact and attacker prerequisites

We aim to acknowledge within 4 business hours, provide initial assessment within 2 business days.

## Disclosure Policy

We follow coordinated disclosure. Public disclosure embargoes can be extended for up to **90 days** if external patching timelines require it.
