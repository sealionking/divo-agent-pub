# Contributing to DiVo Agent Platform

DiVo Agent Platform is currently **source-available (GPLv3)**. This repository serves as the public documentation and distribution front for non-commercial and evaluation use. Source contributions are managed through a CLA process.

If you are interested in contributing, please email `contribute@divo.ai` describing your intended contribution area.

## Community Channels

- Issues: Please use the issue tracker under this repo for documentation errors, typos, and usage questions.
- Feature Requests: Email `contact@divo.ai`. Public voting on features is planned but not yet live.
- Security: See `SECURITY.md` for responsible disclosure.
